Projeto:

Sistema completo para gerenciamento de uma loja de velas artesanais.

Funções:

- Catálogo de produtos
- Sistema de estoque
- Painel administrativo
- Cadastro de produtos
- Upload de imagens
- Configuração da loja
- Sistema de avaliações
- Controle de disponibilidade

Linguagens:

- HTML
- CSS
- JavaScript
- PHP
- MySQL

Objetivo:

Projeto desenvolvido para simular uma solução real para uma pequena empresa, com foco em gerenciamento simples e experiência do cliente.
